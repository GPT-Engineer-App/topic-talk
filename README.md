# topic-talk

A forum where users should be able to start threads about a topic. A user should also be able to search and filter topics to be able to join in and comment on already ongoing threads.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/topic-talk.git
cd topic-talk
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
